🚀 AI Powered LMS Platform

An advanced AI-Powered Learning Management System (LMS) built using the MERN Stack with modern SaaS-level features like AI Smart Search, Google Authentication, Online Payments, and dedicated dashboards for Students & Instructors.

This project is designed to provide a complete real-world learning platform experience with scalable architecture and modern UI/UX.

🌟 Overview

This LMS platform helps students discover courses using AI-powered search and allows instructors to manage and sell courses easily.

It includes:

Secure Authentication
AI Course Search
Payment Integration
Student Dashboard
Instructor Dashboard
Responsive UI
State Management with Redux Toolkit

Perfect for developers who want to build a production-ready SaaS project for their portfolio or resume.
